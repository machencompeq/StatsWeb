We want to determine if there is a significant relationship between customer segment and preferred advertising channel at the 0.05 significance level.
H0 = Customer segment and preferred advertising channel are independent.
Ha = Customer segment and preferred advertising channel are dependent.
Significant level alpha = 0.05.

Plan
We use chi-square test for independence.
Randomness: We assume the 190 customers represent a random sample from the target population.
Independence: The total sample size n = 190 is likely less than 10% of all customers in the population.
Large counts: All expected counts must be >= 5.
Expected count E = (row total * column total) / grand total.
Min expected = (80 * 90) / 190 = 37.89
Since 37.89 >= 5, the condition is satisfied.

Do
Data Summary:
Observed (O):
Millennial/Social Media = 72
Millennial/Email Campaign = 38
Gen X/Social Media = 28
Gen X/Email Campaign = 52
Expected (E):
Millennial/Social Media = (110 * 100) / 190 = 57.89
Millennial/Email Campaign = (110 * 90) / 190 = 52.11
Gen X/Social Media = (80 * 100) / 190 = 42.11
Gen X/Email Campaign = (80 * 90) / 190 = 37.89
chi-square = sum((Oi - Ei)^2 / Ei)
= (72 - 57.89)^2 / 57.89 + (38 - 52.11)^2 / 52.11 + (28 - 42.11)^2 / 42.11 + (52 - 37.89)^2 / 37.89 = 17.24
df = (r - 1)(c - 1) = (2 - 1)(2 - 1) = 1
Critical value chi-square = 3.84146 at alpha = 0.05.

Conclude
Since the calculated test statistic chi-square = 17.24 is greater than the critical value 3.841, it falls into the rejection region. We reject H0 at alpha =0.05. There is sufficient evidence at the 0.05 level of significance to conclude that there is a significant relationship between customer segment and preferred advertising channel at alpha = 0.05.
